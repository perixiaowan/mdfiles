# `Pycharm`中如何安装软件包

- `File`>`settings`>`Project:pythonfile`>`Project Interpreter`
- `Project Interpreter`:选择python解释器
- 选择`+`按钮
- 搜索需要安装的包
- 点击`Install Package`即可
